# SPEC PRPの実行

既存のSPEC PRPを使用して仕様を実装します。

## PRPファイル: $ARGUMENTS

## 実行プロセス

1. **仕様を理解**
   - 現状分析
   - 目標状態の目標
   - タスク依存関係

2. **ULTRATHINK**
   - 計画を実行する前によく考えてください。すべての要件に対処する包括的な計画を作成します。
   - todosツールを使用して、複雑なタスクをより小さく管理可能なステップに分解します。
   - TodoWriteツールを使用して、実装計画を作成し追跡します。
   - 従うべき既存コードから実装パターンを特定します。

3. **タスクを実行**
   - タスク順序に従う
   - 各タスク後に検証を実行
   - 続行前に失敗を修正

4. **変換を検証**
   - 目標状態が達成されたことを確認
   - すべての検証ゲートを実行
   - 統合をテスト

各目標を体系的に進めます。
